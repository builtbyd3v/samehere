## Deferred — Group posting

**Idea:** `post_collaborators(post_id, user_id, role)` + composer UI to invite one co-author. Post shows both avatars; both earn heatmap credit on publish.

**Why defer:** Needs RLS on who can attach collaborators, notification on invite, and abuse controls. Better as v1.5 after DMs exist.

## Deferred — Direct messaging

**Idea:** `conversations` + `messages` tables, `.edu-only`, thread list + chat view, optional request/accept for non-mutuals.

**Why defer:** CLAUDE.md explicitly defers DMs. Biggest gap in interaction today (follow + comment only). Recommend v1.5 flagship alongside notifications realtime.

**Minimum viable DM:** 1:1 text only, no media, block-aware, report message, no read receipts in v1.
